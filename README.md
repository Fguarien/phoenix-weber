# Projet Phénix — Weber Spirit E310 (2007)

Remise en état complète d'un Weber Spirit E310 de 2007.

## Arborescence

```
phoenix-weber/
├── 01-etat-initial/          état de départ (photos + constat)
│   ├── photos/
│   └── constat/
├── 02-diagnostic/            pièces manquantes, usées, à remplacer
│   ├── schemas/              vues éclatées annotées
│   └── pieces/               listes de pièces (md, csv)
├── 03-commandes/             historique achats
│   ├── factures/             PDFs factures reçues
│   └── confirmations/        emails/PDF confirmations commande
├── 04-fournisseurs/          fiches fournisseurs + contacts
├── 05-docs-techniques/       documentation officielle Weber
│   ├── manuels/              manuels PDF
│   └── schemas-eclatés/      vues éclatées constructeur
├── 06-travaux/               suivi chantier
│   ├── etapes/               notes par étape (nettoyage, remplacement…)
│   └── photos-avancement/    photos datées
└── 07-resultat-final/        photos résultat final
    └── photos/
```

## Modèle

- **Appareil :** Weber Spirit E310
- **Année :** 2007
- **Référence grille :** 3 brûleurs, propane/naturel, fonte/acier
